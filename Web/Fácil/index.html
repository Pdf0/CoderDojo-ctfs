Sobre o Desafio

Este desafio foi criado para testar as tuas habilidades de inspeção de código fonte.

Se quiseres inspecionar o código em que a página foi feita podes fazê-lo pressionando:

  • Ctrl + U no Linux ou Windows
  • Cmd + Option + U no Mac
